## Controls
The demo uses the following controls:
  - \<Left Click\> Swap weighted, wall, and floor tiles
  - \<Right Click\> Swap start, end, and floor tiles
  - \<Enter\> Run A* algorithm
  - \<Space\> Clear all tiles
